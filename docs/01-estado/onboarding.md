# Onboarding — Repositorio
- Abrir en Codespaces.
- Si aparece prompt de Antigravity, segur instalación.
- Backend: ver `docs/09-operaciones/local.md`.
- Variables: copiar `.env.example` a `.env`.
- Validar proyecto: `.agents/scripts/validate_project.sh`.
- Leer `AGENTS.md`, `.clinerules`, `.agents/WORLD_STATE.md`.
- Ver `.agents/CONTEXT.md` y `.agents/SKILL_MAP.md` antes de tocar código.
- Frontend: ver `docs/09-operaciones/local.md` para puertos.
- GitHub: ver `.github/workflows/ci.yml`.
