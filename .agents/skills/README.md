# Backend — Skill base
- Regla: toda feature debe traer schema, servicio y ruta.
- Regla: no hardcodear secrets ni usar `shell=True`.
- Regla: toda ruta crítica queda auditada.

# Frontend — Skill base
- Regla: toda página nueva incluye fallback y estado vacío.
- Regla: usar componentes pequeños y tipado estricto.
- Regla: no emojis; usar iconos locales.
